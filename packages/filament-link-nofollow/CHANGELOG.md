# Changelog

All notable changes to `filament-link-nofollow` will be documented in this file.

## 1.0.0 - 2025-11-20

### Added
- Initial release
- Custom TipTap Link extension with rel attribute support
- Dofollow/Nofollow checkbox in link modal
- Target selection (same tab/new tab)
- Automatic rel attribute management
- Edit existing links with pre-filled values
- Artisan command for building assets
- Comprehensive documentation
