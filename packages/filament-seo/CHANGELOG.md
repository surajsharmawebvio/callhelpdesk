# Changelog

All notable changes to `filament-seo` will be documented in this file.

## v1.0.0 - 2025-11-21

### Added
- Initial release
- Polymorphic SEO support for any model
- Complete meta tags (title, description, keywords)
- Open Graph tags for social sharing
- Twitter Card tags
- Schema.org JSON-LD support
- Technical SEO (robots, canonical, sitemap priority)
- Intelligent fallback system
- Performance caching with auto-invalidation
- Filament form component with tabs
- Blade component for meta tag rendering
- SEO Helper with utility functions
- Schema generators (Article, FAQ, Breadcrumb, Organization)
- HasSeo trait for easy model integration
- HandlesFilamentSeo concern for Filament resources
- Comprehensive configuration
- Database migration
- Complete documentation
